# Haltestellen
Mängelmelder zu SPNV Bahnhöfen und Haltepunkten sowie ÖPNV Bus-Haltestellen in Baden-Württemberg

Wenn Sie Fehler in unseren Haltestellen-Daten finden und melden möchten, erstellen Sie gerne in diesem Repository einen Issue mit den Details zur Haltestelle und auf welchem der unten aufgeführten Datensatz Sie sich beziehen.

Wir veröffentlichen Daten zu den Haltestellen: 
* [Haltestellen Baden-Württemberg](https://www.mobidata-bw.de/dataset/haltestellen-baden-wuerttemberg) - Haltestellen mit Bereichen und den einzelnen Steigen, inkl. Geokoordinaten und verschiedenen Namensbezeichnungen
* [Barrierefreiheit an BW-Haltestellen](https://www.mobidata-bw.de/dataset/bfrk-barrierefreiheit-an-bw-haltestellen) - Bushaltestellen mit vielen Details, wie barrierefrei diese sind
* [Barrierefreiheit an BW-Bahnhöfen](https://www.mobidata-bw.de/dataset/bfrk-barrierefreiheit-an-bw-bahnhoefen) - Bahnhöfe mit vielen Details, wie barrierefrei diese sind
